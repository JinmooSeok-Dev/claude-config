현재 세션에서 알게 된 것, 발견한 것, 조사한 핵심 내용을 프로젝트 루트의 `TIL.md`에 기록한다.

## 1. 기존 TIL.md 확인
- 파일이 있으면 끝에 추가 (기존 내용 유지)
- 없으면 새로 생성:
  ```markdown
  # TIL (Today I Learned)
  작업 중 알게 된 것, 발견한 패턴, 참고할 트릭을 기록한다.
  ```

## 2. 엔트리 형식

```markdown
---

### [YYYY-MM-DD] 제목
> 분류: 발견 | 조사 | 트릭 | 패턴 | 이슈해결

**맥락**: 어떤 작업 중에 알게 되었는가 (1줄)

**내용**:
- 핵심 내용 정리
- 코드/명령어 예시 (있으면)

**출처**: [링크 또는 파일:라인] (있으면)
```

## 3. 규칙
- WORKLOG보다 가볍게 — 핵심만 간결하게
- "나중에 다시 찾아볼 때 유용한가?" 기준으로 기록 여부 판단
- 엔트리 초안을 먼저 보여주고 사용자 확인 후 기록한다
