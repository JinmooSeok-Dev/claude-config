## 선형대수 분석 프레임워크

$ARGUMENTS에 대해:

### 1. 문제 유형 파악

| 유형 | 핵심 | 활용 예시 |
|------|------|----------|
| **연립방정식** | Ax = b | 회로 해석, 최적화 제약 |
| **고유값/고유벡터** | Av = λv | PCA, PageRank, 진동 모드 |
| **행렬분해** | A = UΣVᵀ, LU, QR, Cholesky | 차원축소, 압축, 안정적 풀이 |
| **선형변환** | T: V → W | 좌표 변환, CNN 커널, attention |
| **벡터공간** | span, basis, dimension | 특징 공간, embedding |
| **내적/노름** | ⟨u,v⟩, ‖v‖ | cosine similarity, 정규화 |
| **최소제곱** | min ‖Ax - b‖² | 회귀, 곡선 피팅 |
| **텐서** | 다차원 배열 연산 | 딥러닝 텐서 연산, einsum |

### 2. 기하학적 직관

모든 선형대수 개념을 기하학적으로 먼저 설명한다:

- **행렬 곱셈** = 공간의 변환 (회전, 스케일, 전단)
- **고유벡터** = 변환해도 방향이 안 바뀌는 축
- **SVD** = 회전 → 스케일 → 회전 으로 분해
- **rank** = 출력 공간의 실제 차원수
- **determinant** = 변환 후 부피의 배율

```
원래 공간        A 적용 후
  ^  |            /  |
  |  |           /   |  ← 전단(shear)
  |  |          /    |
  +--+-->      +-----+-->
```

### 3. 풀이

**해석적 풀이:**
- 단계별 수식 전개
- 각 단계에서 "이 조작의 의미는 ..."

**수치적 풀이:**
```python
import numpy as np

# 고유값 분해
eigenvalues, eigenvectors = np.linalg.eigh(A)

# SVD
U, S, Vt = np.linalg.svd(A)

# 최소제곱
x, residuals, rank, sv = np.linalg.lstsq(A, b, rcond=None)
```

**수치적 안정성 주의:**
- 조건수 cond(A) = σ_max / σ_min 확인
- ill-conditioned 행렬은 정규화 또는 분해 방법 변경

### 4. ML/시스템 연결

개념이 실제 ML/AI 시스템에서 어떻게 쓰이는지 연결한다:
- **Attention**: Q·Kᵀ = 유사도 행렬, softmax 후 V에 선형결합
- **역전파**: 야코비안 행렬의 연쇄 곱
- **PCA**: 공분산 행렬의 고유값 분해
- **LoRA**: 저rank 행렬 분해로 파라미터 효율적 fine-tuning
- **분산 연산**: 행렬 분할 (TP = 열/행 분할, PP = 계층 분할)

## 과정 표시
매 단계마다 아래 형식으로 과정을 보여준다:
```
[관찰] A는 3×3 대칭행렬 → 고유값이 모두 실수, 고유벡터 직교
[방법] 대칭이므로 Spectral Decomposition: A = QΛQᵀ
[단계 1/3] det(A - λI) = 0 → 특성방정식 전개
[단계 2/3] λ₁=5, λ₂=2, λ₃=1 → 각 고유벡터 계산
[해석] λ₁이 지배적 → 이 방향으로 분산이 가장 큼 (PCA의 제1주성분)
[검증] QΛQᵀ를 다시 곱해서 A와 일치하는지 확인
```
- 행렬의 구조적 특성(대칭, 양정치, sparse 등)을 먼저 관찰한다
- 어떤 분해/방법을 쓸지와 그 이유를 밝힌다
- 수식 전개마다 기하학적 의미를 병기한다

## 규칙
- 기하학적 직관 → 수식 → 코드 순서로 설명한다
- 차원이 2-3인 작은 예제로 먼저 보여준다
- 큰 행렬은 구조(sparse, symmetric, positive definite)를 먼저 파악한다
- 수치 계산 시 dtype과 정밀도(fp32/fp16/bf16)를 고려한다
